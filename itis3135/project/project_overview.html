Project Overview

This project is a web-based application developed for my client Nolan, which hosts an AI-powered NFL Draft Predictor. The application allows users to view, analyze, and interact with AI-generated predictions for upcoming NFL drafts. It helps fans, fantasy football players, and analysts explore draft scenarios and player rankings powered by artificial intelligence.

Intended Users

  • Football fans interested in draft predictions
  • Fantasy football participants
  • Analysts and bloggers who cover the NFL Draft
  • Students learning about AI in sports applications

Website Content Overview

  • Home – Welcome message, mission statement, and featured content
  • Draft Predictions – Interactive mock draft viewer
  • About the AI – Explanation of the algorithm used
  • Compare Prospects – Tool to compare player stats side-by-side
  • Contact – Feedback form to reach the client

Page Design Summary

    Users: General visitors
    Content: Hero section, intro to AI, highlights

  • Draft Predictions
    Purpose: Display current draft predictions
    Users: All

    Interactions: AJAX loading, filter dropdowns
  • About the AI
    Purpose: Explain AI logic and data sources
    Users: Tech-savvy users
    Content: Flowchart, text blocks
    Interactions: Tooltips, collapsible sections
  • Compare Prospects
    Purpose: Compare draft prospects
    Users: Advanced users
    Content: Dropdowns, stat tables
    Interactions: Chart.js, jQuery UI
  • Contact
    Purpose: Collect feedback
    Users: Visitors
    Content: Contact form (name, email, message)
    Validations: Required fields, valid email
    Interactions: Form submit, confirmation alert

Dynamic Functionality

  • Draft Predictions: AJAX used to load data dynamically without page refresh
